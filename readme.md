## Getting Started

### Installation

1. Clone, Add and Push

```Shell
   git clone git@github.com:[GITHUB EXAM](https://github.com/Khintin/holidaze-finals)
```

2. Install dependencies

```shell
   npm install
```

3. To access local host and start the project

```shell
   npm run dev
```
